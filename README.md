Overview:
Developed as part of the **Infosys Springboard Virtual Internship**,the **Fake News Detection & Verification Tool** is an AI-powered system designed to combat the growing spread of misinformation across digital platforms. The project leverages **Natural Language Processing (NLP), transformer-based deep learning models, and real-time fact-checking mechanisms** to analyze news articles, identify suspicious claims, and classify content as **real, suspicious, or fake**. By combining automated text preprocessing, entity extraction, and credibility verification against trusted sources, the system provides **explainable results that highlight misleading information and support users with verified references**. This project demonstrates the practical application of **machine learning and AI in addressing real-world challenges such as misinformation detection and digital media credibility.**
